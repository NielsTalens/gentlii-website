# Gentlii Header Text Links Design

## Goal

Remove the button-like styling from the header navigation so the module links read as plain text.

## Chosen Approach

Keep the current sticky full-width header structure and simplify only the link treatment:

- no pill background
- no border
- no rounded link shape
- text color change on hover and focus

## Result

The header keeps its structure and spacing, but the navigation reads as lightweight editorial text instead of controls.
